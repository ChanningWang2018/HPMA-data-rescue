## 描述

扔出一瓶肿胀药水，在范围内形成持续一段时间的药水滩。

经过药水的**友方**单位，在一段时间内体型增大，且攻速和移速增加。

### 备注

肿胀药水的持续时间每级增长0.2秒，其增益效果的持续时间则始终为9秒。


## 故事

肿胀药水是一种由蝙蝠脾脏、干荨麻和河豚眼睛制成的魔药，接触到药水的身体部位会发生肿胀。

曾经为了偷拿斯内普私人储藏室中的复方汤剂原料，哈利在魔药学课堂上点燃了烟火，不料烟火落到了格雷戈里·高尔的坩埚中，被药水溅到的同学身上都发生了肿胀，最后不得不找教授领消肿剂。

## log

[2022-1-26](/blog/220126-update-annc#adjustment)调整：增益效果持续时间由6秒提升到9秒。

[2021-12-1](/blog/211201-update-annc#adjustment)调整：所消耗的魔力值，由3点降低为2点。同时，攻速增加调整为40%，移动速度增加调整为30%，体型增加调整为30%。

设计师观点：目前，肿胀药水作为辅助性魔咒，其所带来的增益效果不能立刻见效，而一旦错过使用时机又会非常吃亏。所以我们调整了肿胀药水的费用及效果，使其能够更加合理的搭配到各个魔咒组合中。
