## 描述

在目标范围释放一个护罩，为护罩内友方单位承受伤害。

*护罩每2秒会损失一定生命值，生命值下降为0时护照消失。

*护罩**本身**免疫控制效果，同时**无法**被治疗且**无法**获得护盾值和增益效果。

## 故事

“统统加护”是一个防御性魔咒，它能召唤出一道屏障保护一片特定的区域或某座建筑，在一定时间内挡下或防御大部分咒语。它可能是“盔甲护身”的一个变种。

赫敏、哈利和罗恩在寻找魂器期间，赫敏使用过统统加护来保护和隐藏他们的栖身之地。在施咒的同时，空间中似乎会产生热气。

[2022-12-7](/blog/221207-update-annc#fix)修复了“愈合如初”对“鲁伯·海格”伙伴卡的摩托车、“统统加护”屏障等无法治疗单位施放的问题。

[2022-8-31](/blog/220831-update-annc#fix)修复了魔咒“统统加护”的特效大小和实际范围不一致的问题。

[2022-5-31](/blog/230601-update-annc#adjustment)调整：每2秒扣血比例由13%调整为20%。

设计师观点：在目前部分对局中，统统加护较长的持续时间常常使巫师难以反制。因此，我们略微提升了统统加护的扣血比例，希望统统加护的释放能够更注重时机的选择。

[2022-4-7](/blog/220407-update-annc#adjustment)[调整]：统统加护调整为选择地面位置释放，并且可以对友方单位释放，护盾值提升10%，同时每2秒扣除的护盾值提升至13.3%。快捷施法时将依旧以自身为中心释放。

[设计师观点]：目前统统加护只能对巫师自身释放，使用起来多有不便。因此，我们调整了统统加护的释放方式，帮助巫师更灵活地使用统统加护保护队友以及己方召唤物。同时，为了平衡统统加护的整体强度，我们提高了统统加护的护盾值，同时略微上调了护盾值的削减速度。


[2021-12-1](/blog/211201-update-annc#adjustment)调整：盔甲的生命值提高6.25%。
